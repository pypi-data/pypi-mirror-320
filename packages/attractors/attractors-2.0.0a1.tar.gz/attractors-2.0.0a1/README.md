<h1>attractors</h1>
attractors is a package for simulation and visualization of strange attractors.
</div>

## Installation

```bash
pip install attractors
```

From source:
```bash
git clone https://github.com/vdesmond/attractors
cd attractors
pip install .
```

## Dependencies
- python >= 3.11
- matplotlib>=3.10.0
- numba>=0.60.0
