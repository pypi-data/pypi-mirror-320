Highlights

This is a stealth bleed & vote deck starring {{ lutz_von_hohenzollern }} and {{ undele }}. It uses {{ zillah_s_valley }} and {{ information_highway }} to get these big capacity vampires in play quickly, then {{ villein }} to get some pool back.

The sheer number of votes the vampire have makes sure they get the vote lock around the table. Stealth is high, with the versatile {{ perfect_paragon }} providing either stealth or votes, depending on what's needed on the table.

The deck is slow but sturdy and applies constant high pressure to its prey.

Tips & Tricks

{{ voter_captivation }} and {{ dmitra_ilyanova }}'s ability can be used to get pool back on the vampires after {{ villein }}.

{{ banishment }} is used to get rid of cumbersome vampires. It can either weaken the prey, slow down the predator, or even be used cross table to maintain the table balance if necessary. Moreover, it can dramatically increase the power of key votes like {{ ancient_influenceÂ }}.

As the game progresses, {{ undele }}'s ability can be used for recursion, getting the cards most adapted to the table context back in hand.

This version uses {{ force_of_personality }} for combat defense, so it can be cycled out of hand easily if it's not needed.

Variants

The Maris Lutz variant uses a group 3-4 crypt with {{ maris_streck }}, like {{ danilo_torrisi_s_deck }} from the 2016 EC. Using more Malkavian in the crypt makes {{ madness_network }} a useful inclusion, and having a bit more Justicar and Princes allows for some {{ parity_shift }} to provide more bloat.

The Helicopter variant uses the {{ alastor }} + {{ helicopter }} combination to enable {{ lutz_von_hohenzollern }} to vote and bleed in the same turn, like in {{ fabien_garcia_s_deck }} from 2014 EC FCQ.

The AD&D variant uses {{ derange }} and {{ malkavian_dementia }}, like {{ otso_saariluoma_s_deck }} from 2010. This combination can steal any vampire, robbing them of their blood and equipment using {{ heidelberg_castle_germany }}, or even destroying them for pool with {{ golconda_inner_peace }}, provided one can get a second Master Action with {{ cybele }} or {{ the_parthenon }}.
