2.2 Левая и правая граница ДИ для математического ожидания
n = Счёт(A:A)
xср = Срзнач(A:A)
S = Стандотклон.В(A:A)
gamma = дано в услов.
alpha = 1 – gamma
t = Стьюдент.Обр(1 – alpha/2;n -1)
delta = (t * s) / корень(n)
Eлев = xср – delta
Eправ = xср + delta

2.3 Левая и правая граница ДИ для дисперсии
n = Счёт(A:A)
s^2 = Дисп.В(A:A)
gamma = дано в услов.
alpha = 1 – gamma
числитель = (n – 1) * s^2 (эта переменная уже есть)
хи2лев = хи2.обр(alpha/2;n-1)
хи2прав = хи2.обр(1 – alpha/2;n-1)
varлев = числитель/хи2прав
varправ = числитель/хи2лев
