
### How to add a new endpoint : 
```

 1) Create the dataclass  in api/datamodels.py
         xxxxRequest
         xxxxOutput 


 2) Add the entrypoint code/interface  in app.py
         def api_xxxx

 3) Add the params validation in api/validation.py
 

 4) Add processing interface in api/process.py
 

 5) If needed, add the computing component in engine/




```
